# Setup & Run — Alley Dining Scraper

## 1. Create & activate virtual environment

```bash
python3 -m venv venv
source venv/bin/activate
```

## 2. Install dependencies

```bash
pip install -r requirements.txt
```

## 3. Run the scraper

```bash
python alley_dining.py
```

Output is saved to `alley_dining_menus.json` in the same directory.

## Deactivate when done

```bash
deactivate
```
