# BuffBites

AI-powered meal combo suggester for CU Boulder dining halls. Give it a dining hall and date, get back 4 creative combos generated by Claude.

---

## Stack

- **Backend:** Node.js, Express
- **AI:** Anthropic Claude (`claude-haiku-4-5`)
- **Data:** Pre-scraped JSON menus in `scraping_scripts/data/`

---

## Run

```bash
cd backend
npm install
node server.js
```

Requires a `.env` file in `backend/`:
```
ANTHROPIC_API_KEY=...
PORT=3001
```

---

## API

`GET /api/combos/generate?dining=<hall>&date=<YYYY-MM-DD>`

Valid `dining` values: `alley`, `c4c`, `libby`, `sewall`, `village_center`

`date` defaults to today if omitted.

**Example:**
```bash
curl "http://localhost:3001/api/combos/generate?dining=c4c&date=2026-03-16"
```

See `API_DOCS.md` for full details.
